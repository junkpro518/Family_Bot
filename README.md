# 🤲 Family Bot — بوت صلة الرحم

بوت تيليجرام يذكّرك بالتواصل مع أقاربك بناءً على جدول ذكي محفوظ في Notion.

## ✨ المميزات

- 📅 **توزيع شهري ذكي** — يوزّع التواصل على أيام الشهر بتباعد متوازن
- 🔔 **تذكير في الظهر** + إعادة كل ساعة في `:45` لين تضغط ✅
- 🌙 **يكمل بكره لو فاتك اليوم** — ما ينساك
- 📊 **يحدث Notion تلقائياً** — تشوف رصيدك من تيليجرام
- 🔐 **Single user** — البوت لك أنت فقط

## 🚀 البدء السريع (محلياً للتطوير)

```bash
# 1. إنشاء بيئة افتراضية
python3 -m venv .venv
source .venv/bin/activate

# 2. تثبيت الاعتماديات
pip install -r requirements.txt

# 3. نسخ ملف البيئة
cp .env.example .env
# عدّل .env وضع التوكنات الجديدة (لا تستخدم القديمة المكشوفة!)

# 4. تشغيل
python -m bot.main
```

```
bot/

```

## ⚠️ تحذير أمني

التوكنات (Telegram + Notion) **يجب** أن تكون في ملف `.env` فقط، ولا تُرفع لـ git. الملف موجود في `.gitignore`.

**التوكنات اللي تم مشاركتها في أي محادثة سابقة لازم تتجدد قبل النشر.**

## 📋 الأوامر في تيليجرام

| الأمر | الوظيفة |
|---|---|
| `/start` | تسجيل المالك (أول مرة) |
| `/add` | إضافة قريب جديد |
| `/list` | عرض كل الأقارب |
| `/edit` | تعديل بيانات قريب |
| `/remove` | حذف قريب |
| `/today` | تذكيرات اليوم |
| `/status` | جدول الشهر الكامل |
| `/help` | شرح الأوامر |

## 📊 المخطط في Notion

البوت يستخدم جدول `Family Bot` بالأعمدة التالية:

| العمود | النوع | الوصف |
|---|---|---|
| `الاسم` | Title | اسم القريب |
| `الأحد` ... `السبت` | Checkbox | الأيام المسموحة للتواصل |
| `عدد مرات التواصل في الشهر` | Number | الهدف الشهري |
| `تم التواصل هذا الشهر` | Number | العداد (يصفّر شهرياً) |
| `أيام الشهر للتواصل` | Text | جدول الشهر، مثل `4,17,25` |
| `تذكير معلق منذ` | Date | وقت بدء التذكير المعلق |

البوت يضيف الأعمدة الناقصة تلقائياً عند أول تشغيل (إن كان عنده الصلاحية).
